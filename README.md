# Mara

Mara is an analysis and plotting tool that takes data from tushare and plots it through matplotlib.

## Usage

1. Prepare the ~/.mararc file:

```yaml
  token: '<tushare token>'
```
2. run mara.py

```bash
	$ python3 mara.py [OPTIONS] [KEYWORD [KEYWORD ...]]
```
Example:

```bash
	# print all basic info
	$ python3 mara.py
```
